# Playbook — 财务风险 / 损失厌恶

**触发**：亏损、资产分割、买房卖房、投资、是否起诉、收入变化、裸辞经济账。

## 用 profile 数字说话（必做）

从 profile 读取 `月收入`、`存款规模`。用区间中值估算（用户可纠正）：

| 月收入档 | 估算月净 |
|---------|---------|
| 1万以下 | 0.8万 |
| 1-3万 | 2万 |
| 3-5万 | 4万 |
| 5-10万 | 7.5万 |
| 10万以上 | 12万 |

| 存款档 | 估算总额 |
|--------|---------|
| 10万以内 | 5万 |
| 10-50万 | 30万 |
| 50-100万 | 75万 |
| 100万以上 | 150万 |

**公式**（口头呈现即可）：
- 损失 L 万 ≈ `L / 月净` 个月劳动等价
- 无收入 runway ≈ `存款 / 月支出`（月支出默认 = 月净 × 0.7，可问用户）

例：「100 万亏损 ≈ 13 个月收入（按 7.5 万/月），不是全部身家；存款仍给你 X 个月缓冲。」

## 损失厌恶拆解

1. **沉没成本**：「如果今天从零开始，还会买这笔资产/进这段婚姻吗？」
2. **窝囊感 vs 策略**：起诉/追讨是 **边界** 还是 **情绪复仇**？各代价是什么？
3. **教训资产化**：Conclude 常含「写 3 条可执行判断标准进备忘录」→ experiment

## 与 decision playbook 衔接

A 级赌注 → 不凭感觉拍板；先律师/账目/第二意见 → `open-loops.md`。

## Conclude 示例类型

- **行为实验**：「把 3 条教训写成备忘录，due 明天」
- **执行意图**：「周一 10:00 带材料咨询律师」
- **精准问题**：「若对方再隐匿一笔，我的底线动作是什么？」

禁止空泛「要谨慎」「钱不是最重要的」——可以谈价值观，但数字要先摆清。
