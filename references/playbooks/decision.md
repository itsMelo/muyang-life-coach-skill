# Playbook — 重大决策

**触发**：换工作/城市、婚姻、大额投资、诉讼策略、创业 all-in、是否起诉等不可逆或高赌注选择。

## 前置

Read `profile.md` → `## 关键人生决策` 全表。Read `references/playbooks/finance-risk.md` 若涉及钱。

## 决策脚手架

### 1. 赌注分级

问：「这个决定失败后的 **最坏可量化损失** 是什么？恢复需要多久？」

分档：
- **A 人生级**（>1 年收入或不可逆关系/健康）
- **B 项目级**（3–12 个月收入或 1–2 年窗口）
- **C 实验级**（<1 个月收入或可撤回）

若用户把 C 当 A 感受 → 用 profile 数字去灾难化（见 finance-risk）。

### 2. 同构对比（仅真匹配时）

「你当初 [决策名] 时，选项是 [X]，你选了 [Y]，考量是 [Z]。现在的 [当前选项] 和那时哪里像、哪里不同？」

不同点必须写出来（例如：当时 all-in，现在有 runway）。

### 3. 价值观排序

冲突时：「[价值观 A] 和 [价值观 B] 都重要——哪个是 **方向**，哪个是 **手段**？」对照 `最核心价值观（冲突时不让步）`。

### 4. 不可逆检查清单

- 有没有 **48 小时** 内不必拍板的信息还能拿？（律师、体检、报价）
- 有没有「先小规模试」版本？
- 谁在受影响、你是否已和他们对齐？

### 5. Conclude 默认

优先：**行为实验** 或 **执行意图**（预约咨询、列选项表、小规模试），而非当场拍板——除非用户决心 ≥8/10 且赌注已分级为 C 或信息已齐。

写入 `experiments.yaml`；重大待定项同步 `open-loops.md`。
