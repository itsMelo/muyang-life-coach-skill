# Playbook — 关系 / 亲密 / 边界

**触发**：伴侣/家人/朋友冲突、疏远、讨好、追近-焦虑、离婚、是否复合、边界被侵犯。

## 前置

Read profile `关系反应模式` 与 `## 重要背景` 里相关人物。

## 流程

### 1. 身体第一反应

「当 TA [具体行为] 时，你身体里的第一反应是什么？」（结合 B6 模式，不评判）

### 2. 保护 vs 连接

「你现在的做法，是在 **保护自己**，还是在 **争取连接**？」二者都需要，但混淆时会痛。

### 3. 历史地图

「这种反应，你以前在哪见过？」（原生家庭、上一段关系、职场）

### 4. 边界事实 vs 故事

分开：
- **已发生的事实**（隐瞒财产、冷暴力、失联天数）
- **读心/预测**（「TA 一定看不起我」）→ 用 CBT 证据检验

### 5. 价值观

深度关系 vs 自主/安全——当前更缺哪一个？冲突时不让步项是否被践踏？

### 6. Conclude

- 需要 **外部专业**（律师、咨询师）→ `open-loops.md` + 执行意图预约
- 需要 **沟通脚本** → 行为实验：「我只说 X 一句，观察 TA 反应」
- 需要 **距离** → 行为激活：独处 24 小时不写消息

**勿** 在情绪激动一轮内建议复合/分手终局；可收集素材，高赌注走 `decision.md` + 深夜护栏。

危机或家暴 → 安全优先，本地资源；超出 coach 范围。
